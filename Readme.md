# Task Mangement APP

you can create, edit and delete task

## DB set up

in the root folder of project
```
docker compose run
```

config env file of backend
```
DATABASE_USER=postgres
DATABASE_PASS=postgres
DATABASE_NAME=task
DATABASE_HOST=localhost
```

## Backend

### `cd backend`

### `npm start`


## Frontend

### `cd frontend`

### `npm start`
